🎯 一句话结论

生活小事用 AI，按月计费基本够用，不用算 token。真正的分叉只有一个问题——你有没有「海外条件」：现成的 Claude Code / Codex 订阅，或一个梯子？有 → 自己电脑直接跑；没有 → 国内租个小服务器，用 Claude Code 桥接国产模型（智谱 GLM、DeepSeek V4）。丰俭由人，一个月 ¥0 到一百出头。

🛠️ 这套「随身 AI」怎么搭起来

说白了就三步，花钱的地方只有一处——那个「AI」从哪来。

1️⃣ 跟 AI 说需求 —「帮我把冰箱这些菜配成今晚的饭」「整理一份天津两日游」。
2️⃣ 它生成 HTML 报告、顺手推到 GitHub — 一个能自己动手的 AI agent（Claude Code / Codex）干这活。
3️⃣ 手机小程序打开、转发 — 就是你现在看的这个查看器。

三步里只有第 2 步的「AI」要花钱。下面就把这一处的账算清。

适合：大多数国内用户——不用梯子、按月一口价、还能手机随时连。
