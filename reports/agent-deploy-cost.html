在手机上用 AI 打理生活，一个月要花多少钱？
你在小程序里看到的这些报告——今晚吃什么、赏花图鉴、城市攻略——都是对着一个 AI 说句话生成的。想自己也搭一套「随身 AI」？其实一个月几十块就够。
🇨🇳 面向国内 · 人民币 · 参考价 2026 年中

🎯 一句话结论

生活小事用 AI，按月计费基本够用，不用算 token。真正的分叉只有一个问题——你有没有「海外条件」：现成的 Claude Code / Codex 订阅，或一个梯子？有 → 自己电脑直接跑；没有 → 国内租个小服务器，用 Claude Code 桥接国产模型（智谱 GLM、MiniMax、Kimi、DeepSeek 等）。丰俭由人，一个月 ¥0 到一百出头。

🛠️ 这套「随身 AI」怎么搭起来

说白了就三步，花钱的地方只有一处——那个「AI」从哪来。

1️⃣ 跟 AI 说需求 —「帮我把冰箱这些菜配成今晚的饭」「整理一份天津两日游」。
2️⃣ 它生成 HTML 报告、顺手推到 GitHub — 一个能自己动手的 AI agent（Claude Code / Copilot）干这活。
3️⃣ 手机小程序打开、转发 — 就是你现在看的这个查看器。

三步里只有第 2 步的「AI」要花钱。下面就把这一处的账算清。

🌉 两条路，看你有没有「海外条件」

有没有现成的海外订阅、或者一个梯子（VPN），决定了你走哪条路——两条都能跑起来。

有条件
有订阅 / 有梯子 → 自己电脑直接跑
≈ ¥0–150 / 月 用你已有的就行
  • 有 Claude Code / Codex（ChatGPT Plus）订阅：本机装好直接开跑，不额外花钱。
  • 只有梯子没订阅：本机装 Claude Code，充点海外 API，或订一个 Claude Pro（≈ ¥144/月）。
  • 轻度使用：Copilot Free 之类的免费额度也够起步。
适合：已经会翻墙、或本来就有 ChatGPT / Claude 会员的人——别折腾，本机最省事。
国内直连
没梯子没订阅 → 国内服务器 + 国产模型桥接
≈ ¥50–90 / 月 一顿饭钱
  • 简版（最省）：本机直接装 Claude Code，把接口地址指向国产 plan——国内直连、不用梯子。¥29–49/月或更少。
  • 随身版（推荐）：再加一台国内轻量服务器（新人 ¥7/月起），手机 SSH 一连就能随时随地喊它干活、24 小时在线，不用把家里电脑一直开着。
  • 模型：国产 coding plan 一口价——MiniMax ¥29 / 智谱 GLM ¥49 / Kimi Code 起，都适配 Claude Code；或 DeepSeek V4 按量（白菜价，用不了几个钱）。
适合：大多数国内用户——不用梯子、按月一口价、还能手机随时连。
